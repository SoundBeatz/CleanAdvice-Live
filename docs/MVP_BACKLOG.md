# MVP backlog

## Goal

Deliver a first usable Clean Advice Live MVP that can run on advice.clean360.nl.

## MVP scope

### Must have

- LiveKit video call app
- LemonSlice avatar
- Secure server-side token route
- Multilingual AI behavior
- Clean Advice service advisor
- Clean Quote intake workflow
- Clean Service basic troubleshooting
- Knowledge lookup from local files
- Conversation summary
- Email or export-ready lead summary

### Should have

- Welcome video screen
- Ringing state
- bot_ready state handling
- Basic Clean360 branding
- Service ticket draft
- Quote request draft

### Later

- Admin dashboard
- Shop integration
- CRM integration
- AI Vision
- Digital Twin
- Predictive maintenance

## First service protocols

- SRV-001: no water pickup
- SRV-002: streaks behind machine
- SRV-003: brush does not rotate

## First floor types

- Rubber floor
- PVC
- Tile
- Concrete
- Epoxy

## First machines

- i-mop Lite
- i-mop XL
- i-mop XXL
- i-vac 6
- i-vac 9

## Definition of done

The MVP is ready when a visitor can open the app, start a video call, talk to Clean Advice in their own language, receive useful advice, and generate a clean quote or service summary.
