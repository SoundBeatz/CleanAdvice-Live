# Clean Knowledge

Clean Knowledge is the central knowledge layer for the Clean360 AI Platform.

Clean Advice is the video front-end. Clean Knowledge is the brain behind it.

## Purpose

Clean Knowledge stores the operational, commercial and technical knowledge that all Clean360 AI modules use.

Modules that use Clean Knowledge:

- Clean Advice
- Clean Quote
- Clean Shop
- Clean Service
- Clean Planner
- Clean Dashboard
- Clean Admin

## Knowledge categories

- Services
- Machines
- Products
- Floor types
- Cleaning methods
- Quote rules
- Service issues
- Troubleshooting
- Safety rules
- Customer profiles
- Building profiles
- Calculation rules

## Design principle

Do not hard-code business knowledge into prompts.

Prompts define behavior.
Knowledge files define facts, services, products, methods and rules.

## Long-term goal

Clean Knowledge becomes the shared AI brain for Clean360, Fit Advice and Sound Advice.

Each brand can have its own knowledge profile while using the same platform engine.
