# Quote intake schema

Clean Quote collects the information needed to prepare a professional Clean360 quote.

## Required fields

- Contact name
- Company name
- Email address
- Phone number
- Location
- Type of building
- Approximate square meters
- Floor types
- Cleaning frequency
- Current problem
- Desired result
- Preferred start date
- Preferred contact method

## Optional fields

- Number of sanitary rooms
- Number of changing rooms
- Gym or sports environment
- Machine interest
- Product interest
- Current cleaning supplier
- Budget indication
- Photos or video

## Output summary

The AI should summarize the quote request clearly before handoff.
