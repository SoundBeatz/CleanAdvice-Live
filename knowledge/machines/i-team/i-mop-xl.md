# Machine DNA - i-team i-mop XL

## Type

Schrobzuigmachine voor harde vloeren.

## Toepassing

Geschikt voor professioneel vloeronderhoud in onder andere:

- sportscholen;
- scholen;
- kantoren;
- winkels;
- zorglocaties;
- horeca;
- openbare gebouwen.

## Belangrijke onderdelen

- Squeegee / vloerwisser
- Squeegee rubbers
- Zuigslang
- Vuilwatertank
- Schoonwatertank
- Vlotter
- Zuigmotor
- Borstels of pads
- Accu
- Lader

## Veelvoorkomende symptomen

- Machine zuigt geen water op
- Machine laat strepen achter
- Machine laat water liggen
- Borstels draaien niet
- Machine geeft weinig water
- Machine maakt veel lawaai
- Accu loopt snel leeg

## Serviceprotocollen

- SRV-001: machine zuigt geen water op

## Praktijknotities

Bij wateropnameproblemen eerst controleren:

1. Squeegee recht en rubbers heel.
2. Zuigslang doorspoelen.
3. Vuilwatertank goed gesloten.
4. Vlotter vrij bewegend.
5. Zuigmotor hoorbaar actief.
