# Floor advice rules

## Rubbervloer

IF floor type is rubber
THEN ask about:

- sports environment;
- square meters;
- sweat, magnesium or odor;
- cleaning frequency;
- current machine;
- current chemical product;
- squeegee condition;
- changing rooms and showers.

IF rubber floor has odor
THEN check cleaning frequency, ventilation, product residue and sanitary zones.

IF rubber floor has black stripes
THEN check contamination type, pads, tire marks, squeegee and product buildup.

## Unknown floor type

IF floor type is unknown
THEN ask for a photo or describe the surface.

## Safety

IF user is unsure about chemical compatibility
THEN avoid strong recommendations and advise testing on a small inconspicuous area first.
